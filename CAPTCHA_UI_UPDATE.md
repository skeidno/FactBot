# 验证码识别界面更新说明

## 🎨 UI 设计更新

根据参考设计图，已将验证码识别界面重新设计为现代化的紫色渐变风格。

## ✨ 主要特点

### 1. 视觉设计
- **紫色渐变背景**: 使用 `linear-gradient(135deg, #667eea 0%, #764ba2 100%)` 创造科技感
- **白色主卡片**: 圆角卡片设计，带阴影效果，提升层次感
- **现代化按钮**: 渐变背景按钮，悬停效果流畅

### 2. 布局结构

```
┌─────────────────────────────────────┐

│  ┌──────────────┐  ┌──────────────┐ │
│  │ ddddocr 引擎 │  │ AntiCAP 引擎 │ │
```

### 3. 组件设计

#### 引擎选择器
- 网格布局（2列）
- 选中状态：紫色渐变背景 + 白色文字
- 未选中状态：浅灰背景 + 灰色文字
- 显示引擎描述信息

#### 类型选择器
- 标题区域：图标 + 标题 + 描述
- 按钮网格：自适应列数（最小 200px）
- 选中状态：紫色渐变
- 支持 ddddocr 标签显示

#### 图片上传器
- 双图布局：需要副图时自动显示网格
- 单图布局：只显示主图上传区域
- 上传区域：虚线边框 + 大图标
- 预览区域：白色卡片 + 移除按钮

#### 识别按钮
- 全宽按钮
- 紫色渐变背景
- 图标 + 文字组合
- 禁用状态：灰色背景

#### 结果显示
- 成功：绿色渐变背景
- 失败：红色渐变背景
- 标题区域：图标 + 状态文字
- 内容区域：等宽字体显示

- 引擎选择：2列网格
- 类型按钮：自适应网格（最小 200px）

- 移动端友好：自动调整布局

### 交互效果
- 按钮悬停：过渡动画 0.3s
- 状态切换：平滑过渡
- 图片预览：圆角 + 阴影
- 加载状态：图标动画

## 📱 用户体验

### 操作流程
1. 选择识别引擎（ddddocr / AntiCAP）
2. 选择验证码类型
3. 上传图片（主图 + 副图）
4. 点击"开始识别"
5. 查看识别结果

### 智能提示
- 副图自动显示：根据验证码类型
- 副图标签：自动显示正确的提示文字
- 按钮状态：未上传图片时禁用
- 加载提示：识别中显示动画

## 🚀 性能优化

- 使用 Signal 管理状态
- 条件渲染减少 DOM
- 图片 Base64 编码
- 异步请求处理
